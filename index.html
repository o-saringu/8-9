プログラム復習コンテンツ

sin(),cos()関数による円運動

サンプルコードの結果

サンプルコード

　　 float angle=0.0;

　　 float offset=60;

　　 float scalar=40;

　　 float speed=0.05;

　　 　

　　 void setup(){

　　 　size(120, 120);

　　 }　

　　 　

　　 　void draw(){

　　 　　float x=offset+cos(angle)*scalar;

　　 　　float y=offset+sin(angle)*scalar;

　　 　　ellipse(x, y, 40, 40);

　　 　　angle+=speed;

　　 　}

説明文

　sin(),cos()関数は正弦（sin）、余弦（cos）を求める関数である。

　

　sin()とcos()を組み合わせると、円運動を作ることができる。

　cos()がx座標の、sin()がy座標の基となる。

　2つの値に変数scalarを掛けて回転の半径を変更し、変数offsetを加えて回転の中心を移動させる。
